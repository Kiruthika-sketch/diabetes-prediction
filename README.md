# Diabetes Prediction using Deep Learning

## Description
This project predicts whether a person has diabetes using a trained deep learning model.

## Features
- Uses pre-trained model  
- Predicts diabetes from input data  
- Simple and fast  

## Technologies Used
- Python  
- TensorFlow / Keras  
- NumPy  

## Files
- model.json → Model structure  
- model.h5 → Model weights  
- test.py → Prediction code  
- pima-indians-diabetes.csv → Dataset  

## How to Run

1. Install required libraries:

pip install tensorflow numpy


2. Run the prediction:

python test.py


## Input
- 8 medical values (like glucose, BMI, age, etc.)

## Output
- 0 → Not Diabetic  
- 1 → Diabetic
